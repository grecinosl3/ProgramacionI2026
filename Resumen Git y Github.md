**1. ¿Qué es GIT?**

Nos sirve para tener un control de versiones en donde nos permite registrar y gestionar cambios en archivos y proyectos



**2. ¿Cuáles son los comandos más importantes y para que sirven?**

git init: es para iniciar un repositorio 

git clone: para copiar un repositorio

git add: para poder agregar archivos o los cambios que realicemos

git commit: nos sirve para guardar cambios.

git push: es para subir los cambios hechos en nuestro repositorio 



**3. ¿Cuál es la diferencia entre Git y Github?**

Git es la herramienta en los editores de código en donde nos sirve para controlar las versiones y GitHub es una plataforma que sirve para alojar repositorio de Git



**4. Enumere los comandos para subir cambios a un repositorio.**

a): git add 

b): git commit

c): git push



**5. ¿Qué es una branch?**

Branch es una rama independiente en donde se desarrollan los cambios sin afectar el proyecto principal.
